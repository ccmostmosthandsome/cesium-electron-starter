# cesium-electron-starter
CesiumJS on Electron getting starter repo

## Using

### Clone
        $ git clone https://github.com/emadehsan/cesium-electron-starter
        
### Install Packages
        $ cd cesium-electron-starter
        $ npm install
        
### Run Electron App
        $ ./node_modules/.bin/electron .
